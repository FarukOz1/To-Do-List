# To-Do List Projesi

Bu proje, modern web geliştirme eğitimim kapsamında hazırladığım, React ve Tailwind CSS kullanarak geliştirdiğim bir yapılacaklar listesi uygulamasıdır.

## Özellikler
* Görev ekleme, silme ve düzenleme (CRUD).
* Görevlere öncelik seviyesi atama (Yüksek, Orta, Düşük).
* Yüksek öncelikli görevler için dinamik uyarı göstergesi.
* Tamamlanan görevlerin takibi.
* Responsive (Mobil uyumlu) tasarım.

# Kurulum
Projeyi yerelde çalıştırmak için:
1. `npm install`
2. `npm run dev`
